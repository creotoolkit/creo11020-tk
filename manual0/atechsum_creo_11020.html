Technical Summary of Changes for Creo 11.0.2.0
The critical and miscellaneous technical changes in Creo Parametric 11.0.2.0 and Creo TOOLKIT are explained in this section. It also lists the new and superseded functions for this release.

Miscellaneous Technical Changes
The following changes in Creo Parametric 11.0.2.0 can affect the functional behavior of Creo TOOLKIT. PTC does not anticipate that these changes cause critical issues with existing Creo TOOLKIT, Creo Elements/Pro TOOLKIT, or Pro/TOOLKIT applications.
Support for the New Dimension Type
In Creo Parametric 11.0.2.0 and later, the value PRODIMTYPE_IPAR_INT is added to the enumerated data type ProDimensiontype. You can now retrieve the IPAR_INT dimension type of a user-defined feature that includes patterns.
