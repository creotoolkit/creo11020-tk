Element Tree: Boundary Blend
This section introduces and shows how to create, redefine and access boundary blend feature in Creo TOOLKIT.
Overview about Boundary Blend
Use the Boundary Blend tool to create a surface with boundary blend geometry. Create or reference an entity to define the surface in one or two directions. The first and last entity in each direction defines the surface boundaries. You can define more fully the shape of the boundary blend geometry by adding more reference entities.
The Element Tree for Boundary Blend Feature
The element tree for boundary blend feature is documented in the header file Probbl.h.
