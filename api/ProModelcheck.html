Object ProModelcheck

ModelCHECK

ModelCHECK is an integrated application that runs transparently within Creo Parametric and can be executed from an external application.This object provides functions to register and manage custom checks through ModelCHECK.

ProModelcheck Functions:

ProModelcheckCheckFunction(), ProModelcheckCheckRegister(), ProModelcheckCleanupFunction(), ProModelcheckExecute(), ProModelcheckUpdateFunction()
