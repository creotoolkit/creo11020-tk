Object ProPlotdriver

Function ProPlotdriverExecute


Description

Synopsis

ProName interf_name
/* (In)
The name of the interface. Cannot be NULL.
*/

*/
)
Returns

PRO_TK_E_NOT_FOUNDInterface with this name not found.
