Category Mechanica, Object ProMechinterface

Function ProMechcontactinterfacedataAnglebetweensurfacesExprGet


Description
Obtains the angle between surfaces for selection filter tolerance while creating runtime contacts between surfaces in a component-component type interface. Use ProMathExpressionEvaluate() to evaluate it.
