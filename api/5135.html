Category Custom user interface, Object ProUIOptionmenu

Function ProUIOptionmenuSizeSet


Description
Set the size of the optionmenu. This field is used only if the parent is a drawing area. If the width or the height parameter is passed a value smaller than the optionmenu's minimum size, the function will fail. The optionmenu's minimum size can be retrieved using ProUIOptionmenuMinimumsizeGet.
Synopsis

/* (In)
