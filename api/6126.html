Category 2D interface, Object Pro2dExport

Function Pro2dExport


Description
Exports 2d models to a file. The following output formats are supported:
  • PRO_STEP_FILE
  • PRO_IGES_FILE
  • PRO_MEDUSA_FILE
  • PRO_DXF_FILE
  • PRO_DWG_FILE
  • PRO_CGM_FILE
  • PRO_STHENO_FILE
  • PRO_SNAP_TIFF_FILE
Note that the output can be controlled by a combination of the input argument 'data' and several Creo Parametric configuration options related to entity type export options.
Synopsis
#include <ProUtil.h>

/* (In)
The type of file to be exported.

)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSIllegal import file type.
PRO_TK_BAD_CONTEXTThe model was not a 2D model.
PRO_TK_NO_LICENSEThere is no license for this import option.
PRO_TK_OBSOLETE_FUNCThe file format selected is no longer supported on this platform.
PRO_TK_BAD_CONTEXTThe function is not available in Design Test Mode.
