Function ProDimensionOrdinatereferencesSet

Assigns the geometric reference for the ordinate dimension.

ProDimension* dimension

*/

*/
ProDimSense* dimension_sense
/* (In)
Dimension sense used for the attachment.
*/
)
Returns
