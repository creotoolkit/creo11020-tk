Object ProMenubarmenu

Menubar menu

A menu on the menubar, such as the File menu, or a sub-menu under such a menu. In Creo Parametric TOOLKIT, you can add push button, check box, radio group and new menus to the menubar and menus on it.

ProMenubarmenu Functions:

ProMenubarmenuChkbuttonAdd(), ProMenubarmenuChkbuttonValueGet(), ProMenubarmenuChkbuttonValueSet(), ProMenubarmenuMenuAdd(), ProMenubarmenuPushbuttonAdd(), ProMenubarmenuRadiogrpAdd()
