Category Creo Parametric fundamentals, Object ProMdl

Function ProMdlLayerGet


Description
Returns the handle of a layer, given its name and owner.
Replacement in Object TOOLKIT: pfcModelItemOwner::GetItemByName
Synopsis
#include <ProLayer.h>
ProErrorProMdlLayerGet(
ProMdl owner
/* (In)
The model that contains the layer
*/
ProName layer_name
/* (In)
The name of the layer
*/
ProLayer* layer
/* (Out)
The layer
*/
)
Returns
PRO_TK_NO_ERRORThe function successfully retrieved the layer.
PRO_TK_BAD_INPUTSOne or more of the input arguments are invalid.
PRO_TK_E_NOT_FOUNDThe specified layer_name does not exist in owner.
