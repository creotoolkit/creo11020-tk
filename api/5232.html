Category Custom user interface, Object ProUIPushbutton

Function ProUIPushbuttonPositionSet


Description
Set the position to the pushbutton with respect to its parent. This field is used only if the parent is a drawing area.
Synopsis
#include <ProUIPushbutton.h>
ProErrorProUIPushbuttonPositionSet(
char* dialog
/* (In)
The name of the dialog that contains pushbutton.
*/
char* component
/* (In)
The name of the pushbutton component.
*/
int x
/* (In)
X value with respect to the upper left corner of the parent.
*/
int y
/* (In)
Y value with respect to the upper left corner of the parent.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
