Category Windchill servers, Object ProServer

Function ProServerObjectStatusGet


DEPRECATED: Since Wildfire 5.0
SUPERSEDED BY: ProServerObjectIsModified
Description

Note: This function is deprecated. Use ProServerObjectIsModified() instead.

Returns the status of the given object in the workspace.
Replacement in Object TOOLKIT: pfcServer::IsObjectCheckedOut
pfcServer::IsObjectModified

*/

Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSOne or more arguments was invalid.
