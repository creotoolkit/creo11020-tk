Object ProGtol

Geometric tolerance

Superobjects:

ProAnnotation

This object is an attribute of the following objects:

ProAnnotationelem

A geometric tolerance is the maximum allowable deviation from the exact size and shape specified in the model design. You can attach a geometric tolerance to dimensions (reference, driven, radius, or diameter), set datums, single or multiple edges, or another geometric tolerance.
