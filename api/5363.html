Category Custom user interface, Object ProUISlider

Function ProUISliderAnchorSet


Description
Sets the position of the slider with respect to a given anchor location. This function is applicable only if the parent of the slider is a drawing area.
Synopsis
#include <ProUISlider.h>
ProErrorProUISliderAnchorSet(
char* dialog
/* (In)
The name of the dialog that contains the slider.
*/
char* component
/* (In)
The name of the slider component.
*/
int x
/* (In)
X value with respect to the upper left corner of the parent.
*/
int y
/* (In)
Y value with respect to the upper left corner of the parent.
*/
ProUIAnchor anchor
/* (In)
Location on the slider used to set the position.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
