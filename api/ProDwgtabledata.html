Object ProDwgtabledata

Drawing table data

Information required to build a drawing table. This object has to be allocated and filled before creating the table. Stores information about the origin, cells, and row and column properties of a drawing table.

ProDwgtabledata Functions:

ProDwgtabledataAlloc(), ProDwgtabledataColumnsSet(), ProDwgtabledataOriginSet(), ProDwgtabledataRowsSet(), ProDwgtabledataSizetypeSet()
