Category Custom user interface, Object ProUIDialog

Function ProUIDialogHorzposoffsetSet


Description
Set the horizontal offset to the dialog relative to the existing dialog. This function should be used only if the horizontal positioning method of the dialog is PROUIPOSITIONING_RELATIVE.
Synopsis
#include <ProUIDialog.h>
ProErrorProUIDialogHorzposoffsetSet(
char* dialog
/* (In)
Name of the dialog to be set.
*/
int value
/* (In)
Horizontal offset value.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
