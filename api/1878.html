Object ProModel

Callback ProModelRenamePostAction


Description
This is the notification function called after renaming a model.

Users must supply this function by calling ProNotificationSet() with the notify type PRO_MODEL_RENAME_POST.
Synopsis
#include <ProMdl.h>
ProError(*ProModelRenamePostAction)(
ProMdl model
/* (In)
The data of the renamed model
*/
ProMdlFileName old_name
/* (In)
The old name of the model that was renamed
*/
)
Returns
Return values are not interpreted by Creo Parametric.
See Also
ProNotificationSet
