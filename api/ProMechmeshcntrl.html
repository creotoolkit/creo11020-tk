Object ProMechmeshcntrl

Mechanica mesh control

Superobjects:

ProMechitem

Pro/Mechanica mesh controls define the limitations and rules to be applied when generating a mesh, such as the minimum or maximum size of the elements, the distribution of nodes along edges, hard points and hard curves, mesh ID numbering, mesh ID offsets, and the displacement coordinate system.
