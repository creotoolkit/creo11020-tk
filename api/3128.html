Category Manufacturing, Object ProMfgdb

Function ProMfgdbMatchParamAdd


Description
Adds a parameter to the match in the structure allocated by ProMfgdbMatchAlloc().
Synopsis
#include <ProMfgdb.h>
ProErrorProMfgdbMatchParamAdd(
ProMfgdbMatch mfgdb_match
/* (In)
match with parameter added.
*/
ProName param_external_name
/* (In)
parameter name; can be translated. Can be NULL. param_internal_name used if there is no translation.
*/
ProCharName param_internal_name
/* (In)
parameter name; never translated.
*/
ProLine value_wstr
/* (In)
parameter value.
*/
ProParamvalueType value_type
/* (In)
value type ( PRO_PARAM_DOUBLE, PRO_PARAM_STRING...).
*/
)
Returns
PRO_TK_NO_ERRORThe function successfully added the parameter to the match.
PRO_TK_BAD_INPUTSOne or more of the input arguments are invalid.
