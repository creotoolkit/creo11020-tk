Category Custom user interface, Object ProUITree

Function ProUITreeTreecellselectActionSet


Description
Set the action function to be called when the cell of the attribute window of the tree has been selected.
Synopsis

*/
ProUIAction callback
/* (In)

*/
ProAppData data

Action data passed to the callback function. Can be NULL.

)
Returns
