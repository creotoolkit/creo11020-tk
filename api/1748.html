Category Mechanism Design, Object ProKinDrag

Function ProKinDragSessionInquire


Description
this function checks if drag session is on. The drag session may exist automatically in some cases: redefine, regeneration, window switch, representation mode chage etc.
Synopsis
#include <ProKinDrag.h>
ProErrorProKinDragSessionInquire(void)
Returns
PRO_TK_NO_ERRORThe drag session is active.
PRO_TK_GENERAL_ERRORThe drag session is closed.
