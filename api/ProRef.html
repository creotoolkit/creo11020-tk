Object ProRef

External reference control

External references are dependencies between a Creo Parametric object (part or subassembly) and information from another object not owned by either the referencing object or its submodels. Creo Parametric TOOLKIT handles these references with external reference control functions.

ProRef Functions:

ProRefCtrlCheckScope(), ProRefCtrlEnvirGet(), ProRefCtrlEnvirSet(), ProRefCtrlSolidGet(), ProRefCtrlSolidSet()
