Category Parameters and relations, Object ProRelset

Function ProRelsetPostregenerationInit


Description
Initializes a ProRelset data structure for post-regeneration relations.

Note: This function complements ProModelitemToRelset, which is not applicable for post-regeneration relations. The obtained post regeneration ProRelset data can be used in ProRelsetRelationsGet, ProRelsetRelationsSet, ProRelsetRegenerate and ProRelsetDelete.
Replacement in Object TOOLKIT: pfcModel::PostRegenerationRelations
Synopsis
#include <ProRelSet.h>
ProErrorProRelsetPostregenerationInit(
ProMdl pro_model
/* (In)
The model
*/
ProRelset* p_relset
/* (Out)
The post regeneration relation set
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_E_NOT_FOUNDThe post regeneration relation set was not found.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
