Object ProMatrix

Transformation matrix

This object is an attribute of the following objects:

ProView

Transformation matrix is used to obtain the location of a point in solid coordinates relative to its location in the drawing coordinate system. This matrix is constructed from the directions of the three coordinate datum axes and the position of the origin.

ProMatrix Functions:

ProMatrixInit(), ProMatrixMakeOrthonormal()
