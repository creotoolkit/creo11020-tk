Object ProDtlsyminstdata

Detail symbol instance data

The following objects are attributes of this object:

ProDtlsymdef, ProDtlvartext

Properties of a detail symbol instance such as its ID, angle, height, attachment type, color, elbow length, leaders, and options for displaying groups. Information about the symbol definition from which the symbol instance has been derived is also contained by this object.
