Category Custom user interface, Object ProUICheckbutton

Function ProUICheckbuttonAnchorSet


Description
Sets the position of the checkbutton with respect to a given anchor location. This function is applicable only if the parent of the checkbutton is a drawing area.
Synopsis
#include <ProUICheckbutton.h>
ProErrorProUICheckbuttonAnchorSet(
char* dialog
/* (In)
The name of the dialog that contains the checkbutton.
*/
char* component
/* (In)
The name of the checkbutton component.
*/
int x
/* (In)
X value with respect to the upper left corner of the parent.
*/
int y
/* (In)
Y value with respect to the upper left corner of the parent.
*/
ProUIAnchor anchor
/* (In)
Location on the checkbutton used to set the position.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
