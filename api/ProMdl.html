Object ProMdl

Model

Subobjects:

ProDrawing, ProLayout, ProMfg, ProSolid

The following objects are attributes of this object:

ProParameter

This object is an attribute of the following objects:

ProModelitem, ProWindow

A model is a top-level object in a Creo Parametric mode. For example, in Part mode, the model is a part; in Assembly mode, the model is an assembly.
