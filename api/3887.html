Category Solids and parts, Object ProSolid

Function ProSolidAnalysisVisit


Description
Visits External Analyses in the Creo Parametric solid model.
Synopsis
#include <ProSolid.h>
ProErrorProSolidAnalysisVisit(
ProSolid solid
/* (In)
Handle to visited solid
*/
ProAnalysisVisitAction visit_action
/* (In)
Callback visit function
*/
ProAnalysisFilterAction filter_action
/* (In)
Callback filter function. If NULL, all items are visited using the action function.
*/
ProAppData appdata
/* (In)
Application data passed to visit and filter callback functions
*/
)
Returns
PRO_TK_NO_ERRORSuccessful completion of filter and visit functions.
PRO_TK_BAD_INPUTSAn input argument is invalid.
PRO_TK_E_NOT_FOUNDNothing to visit
PRO_TK_GENERAL_ERRORFunction failed
OtherReturn value from visit action function that stops ProSolidAnalysisVisit
