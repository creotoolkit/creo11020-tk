Category Features, Object ProFeature

Callback ProFeatureRegenPreAction


Description
This is the notification function called before regenerating a feature.

This function is to be supplied by the user by calling ProNotificationSet() with the notify type PRO_FEATURE_REGEN_PRE.
Synopsis
#include <ProFeature.h>
ProError(*ProFeatureRegenPreAction)(
ProFeature* feat_handle
/* (In)
The handle to the feature about to be regenerated.
*/
ProElement* p_elem_tree
/* (Out)
The new feature definition upon regeneration. The argument will be initialized to NULL (that is, *p_elem_tree = NULL). It will be used only if the callback initializes it.

Note that this argument is reserved for future use. Values passed to this argument are currently ignored by Creo Parametric.
*/
)
Returns
PRO_TK_NO_ERRORProceed with regeneration.
OtherAn error was encountered. Regeneration is aborted.
