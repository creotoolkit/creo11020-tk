Object ProAnimFrame

Animation frame

The following objects are attributes of this object:

ProAnimObj

This object is an attribute of the following objects:

ProAnimMovie

An animation frame contains animation objects in a specified location. The frames can either be fed directly for single animation or be defined as part of an animation movie to be used for batch animation.

ProAnimFrame Functions:

ProAnimframeCreate(), ProAnimframeDelete(), ProAnimframeObjAdd(), ProAnimframeObjRemove(), ProAnimframeObjVisit()
