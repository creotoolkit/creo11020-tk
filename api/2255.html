Category Mechanica, Object ProMechpoint

Callback ProMechpointVisitAction


Description
Visit action for the datum points that are included in a Creo Simulate geometry model.
Synopsis
#include <ProMechGeometry.h>
ProError(*ProMechpointVisitAction)(
ProMechPoint point
/* (In)
The datum points.
*/
ProError error
/* (In)
Error return passed from the filter action.
*/
ProAppData app_data
/* (In)
Application data passed to the function.
*/
)
Returns
PRO_TK_NO_ERRORContinue visiting.
Any other valueStop visiting and pass this value as a return from the visit function.
