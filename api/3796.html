Category Sheetmetal, Object ProSmt

Function ProSmtFeatureBendtableRemove


Description
Sets a sheet metal feature to use the part bend table instead of a feature bend table. Also causes a regeneration.
Synopsis
#include <ProSheetmetal.h>
ProErrorProSmtFeatureBendtableRemove(
ProFeature* feature
/* (In)
The sheet metal feature (must be a bend or wall)
*/
)
Returns
PRO_TK_NO_ERROR
PRO_TK_BAD_CONTEXTThe feature is not a bend or wall.
