Category Feature element trees, Object ProElement

Function ProElementSpecialvalueGet


Description
Returns the pointer representation for the element.
Synopsis
#include <ProElement.h>
ProErrorProElementSpecialvalueGet(
ProElement element
/* (In)
The element.
*/
ProElementSpecialOptions options
/* (In)
Options for how the pointer should be obtained. Reserved for future use: pass NULL for now.
*/
ProAppData* value
/* (Out)
The pointer value.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSOne or more arguments is invalid.
PRO_TK_INVALID_TYPEThe element does not contain a value that can be represented as a this type.
