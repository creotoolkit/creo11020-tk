Category Custom user interface, Object ProUILayout

Function ProUILayoutAnchorSet


Description
Sets the position of the layout with respect to a given anchor location. This function is applicable only if the parent of the layout is a drawing area.
Synopsis
#include <ProUILayout.h>
ProErrorProUILayoutAnchorSet(
char* dialog
/* (In)
The name of the dialog that contains the layout.
*/
char* component
/* (In)
The name of the layout component.
*/
int x
/* (In)
X value with respect to the upper left corner of the parent.
*/
int y
/* (In)
Y value with respect to the upper left corner of the parent.
*/
ProUIAnchor anchor
/* (In)
Location on the layout used to set the position.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
