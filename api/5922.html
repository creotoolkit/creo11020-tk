Category Custom user interface, Object ProUITree

Function ProUITreeColumnCreate


Description
Create a column of the given name in the attribute window of the tree. The name must be unique within the scope of the columns of the tree.
Synopsis
#include <ProUITree.h>
ProErrorProUITreeColumnCreate(
char* dialog
/* (In)
The name of the dialog that contains the tree.
*/
char* component
/* (In)
The name of the tree component.
*/
char* name
/* (In)
Name of the attribute window in which column should be added.
*/
wchar_t* title
/* (In)
Title that should be displayed above the column.
*/
int width
/* (In)
Width of the column in character widths.
*/
ProUITreeColumnJustification justify
/* (In)
Left/Right/Center justification.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
