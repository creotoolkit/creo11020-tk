Object ProPopupmenu

Popup menu

Popup menus are created at runtime in Creo Parametric and only when the required menus exist in the active window. These functions enable you to access and modify popup menus.

ProPopupmenu Functions:

ProPopupmenuAccessFunction(), ProPopupmenuButtonAdd(), ProPopupmenuCascadebuttonAdd(), ProPopupmenuCreatePostAction(), ProPopupmenuDestroyPreAction(), ProPopupmenuIdGet(), ProPopupmenuNameGet()
