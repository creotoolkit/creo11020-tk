Category Custom user interface, Object ProUIProgressbar

Function ProUIProgressbarPositionSet


Description
Set the position to the progressbar with respect to its parent. This field is used only if the parent is a drawing area.
Synopsis
#include <ProUIProgressbar.h>
ProErrorProUIProgressbarPositionSet(
char* dialog
/* (In)
The name of the dialog that contains progressbar.
*/
char* component
/* (In)
The name of the progressbar component.
*/
int x
/* (In)
X value with respect to the upper left corner of the parent.
*/
int y
/* (In)
Y value with respect to the upper left corner of the parent.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
