Object ProCsys

Datum coordinate system

This object is an attribute of the following objects:

ProGeomitem

A datum coordinate system is a reference feature that can be added to a part or assembly. Creo Parametric always displays coordinate systems with an X-, Y-, and Z-axis.

ProCsys Functions:

ProCsysDataGet(), ProCsysFilterAction(), ProCsysIdGet(), ProCsysInit(), ProCsysToGeomitem(), ProCsysVisitAction()
