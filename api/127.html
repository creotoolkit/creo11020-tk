Category 3D drawings and annotations, Object ProAnnotationelem

Function ProAnnotationelemAutopropagate


Description
Automatically propagates annotation elements within the current model to the currently selected feature based on the standard Creo Parametric algorithm. The behavior is based on the current contents of the selection buffer.
Licensing Requirement
TOOLKIT for 3D Drawings
Synopsis
#include <ProAnnotationElem.h>
ProErrorProAnnotationelemAutopropagate(void)
Returns
PRO_TK_NO_ERRORAnnotation elements were successfully propagated to the selected feature.
PRO_TK_BAD_CONTEXTThe current selections do not allow propagation of annotation elements.
