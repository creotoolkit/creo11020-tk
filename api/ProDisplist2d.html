Object ProDisplist2d

Replacement in Object TOOLKIT: DisplayList2D

2-D display list

The 2-D display list contains the two-dimensional projections of the 3-D display list onto the current window. It is rebuilt from the 3-D display list when the orientation of the solid changes. This list is used to display two-dimensional entities, such as drawings and sections.
