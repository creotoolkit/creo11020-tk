Category Custom user interface, Object ProUIList

Function ProUIListFocusoutActionSet


Description
Sets the focus out action for a list. This function is called when the user moves the cursor off of the list using the mouse or [TAB] key.
Synopsis
#include <ProUIList.h>
ProErrorProUIListFocusoutActionSet(
char* dialog
/* (In)
The name of the dialog that contains the list.
*/
char* component
/* (In)
The name of the list component
*/
ProUIAction callback
/* (In)
The action to perform when the list loses user focus.
*/
ProAppData data
/* (In)
Application data to be passed to callback function
*/
)
Returns
PRO_TK_NO_ERRORThe function was successful
PRO_TK_GENERAL_ERRORThe function failed
PRO_TK_BAD_INPUTSAn invalid argument was given
