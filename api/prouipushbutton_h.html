    Purpose: Sets the position of the pushbutton with respect to a given anchor

        anchor - Location on the pushbutton used to set the position.

    Output Arguments:
        none

    Return Values:

        PRO_TK_BAD_INPUTS - One or more input arguments was invalid.

    int                 height);
