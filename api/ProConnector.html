Object ProConnector

Connector

Superobjects:

ProAsmcomppath

A connector is a specific type of component in a cabling assembly used to identify locations where a cable can be attached. A connector can use two additional parameters; GENDER (MALE or FEMALE) at the component level and ENTRY_PORT at the pin level.
