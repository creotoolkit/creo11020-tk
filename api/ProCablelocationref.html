Object ProCablelocationref

Cable Location Reference

Creo Parametric TOOLKIT functions for accessing the reference details of a cable location. The first location can reference a point on an assembly surface or can be attached to an entry port on a connector.

ProCablelocationref Functions:

ProCablelocationrefAlloc(), ProCablelocationrefFree()
