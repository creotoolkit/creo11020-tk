Object ProDtlitem

Detail item

Superobjects:

ProModelitem

Subobjects:

ProDtlentity, ProDtlgroup, ProDtlnote, ProDtlsymdef, ProDtlsyminst

A detail item is a non-dimensional informative element that can be added to a drawing. Types of detail items available are draft entities, textual notes, symbol definitions, symbol instances, and draft groups.
