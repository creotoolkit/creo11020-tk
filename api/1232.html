Category Feature element trees, Object ProElement

Function ProElementReferencesGet


Description
Returns an array of reference values for the element.
Synopsis
#include <ProElement.h>
ProErrorProElementReferencesGet(
ProElement element
/* (In)
The element.
*/
ProElementReferenceOptions options
/* (In)
Options for how the references should be obtained. Reserved for future use: pass NULL for now.
*/
ProReference** references
/* (Out)
The reference array (a ProArray).
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSOne or more arguments is invalid.
PRO_TK_INVALID_TYPEThe element does not contain a value that can be represented as an array of references.
PRO_TK_EMPTYThe element does not contain references.
