Object ProExtdata

Replacement in Object TOOLKIT: ExternalData

External data

External data associated with a Creo Parametric model defined by the Creo Parametric TOOLKIT application. This data gets stored in the same file as the Creo Parametric model. The external data for a particular model is broken down into classes and data slots.

ProExtdata Functions:
