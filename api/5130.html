Category Custom user interface, Object ProUIOptionmenu

Function ProUIOptionmenuFocusinActionSet


Description
Sets the focus out action for an optionmenu. This function is called when the user moves the cursor onto the optionmenu using the mouse or [TAB] key.
Synopsis
#include <ProUIOptionmenu.h>
ProErrorProUIOptionmenuFocusinActionSet(
char* dialog
/* (In)
The name of the dialog that contains the optionmenu.
*/
char* component
/* (In)
The name of the optionmenu component
*/
ProUIAction callback
/* (In)
The action to perform when the optionmenu gains user focus.
*/
ProAppData data
/* (In)
Application data to be passed to callback function
*/
)
Returns
PRO_TK_NO_ERRORThe function was successful
PRO_TK_GENERAL_ERRORThe function failed
PRO_TK_BAD_INPUTSAn invalid argument was given
