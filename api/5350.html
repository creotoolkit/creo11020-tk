Category Custom user interface, Object ProUISlider

Function ProUISliderDisableTracking


Description
Disables tracking for a slider component. If the slider is tracking the update action is called for each change in position.
Synopsis
#include <ProUISlider.h>
ProErrorProUISliderDisableTracking(
char* dialog
/* (In)
The name of the dialog that contains the slider
*/
char* slider
/* (In)
The name of the slider component
*/
)
Returns
PRO_TK_NO_ERRORThe function was successful
PRO_TK_GENERAL_ERRORThe function failed
