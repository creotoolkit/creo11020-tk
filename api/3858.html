Category View manager, Object ProSimprepdata

Function ProSimprepdataFree


Description
Frees the memory occupied by the ProSimprepdata structure.

NOTE:

Operates on assemblies only, not parts.

Synopsis

ProSimprepdata** p_data

*/
)
Returns
