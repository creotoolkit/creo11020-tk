Object ProGeomitemdata

Function ProGeomitemdataGet


Description
Retrieves the geometric representation data of the specified geometry item.

The following types of Geomitems are supported:

  • PRO_EDGE
  • PRO_EDGE_START
  • PRO_EDGE_END
  • PRO_CURVE
  • PRO_CRV_START
  • PRO_CRV_END
  • PRO_AXIS
  • PRO_SURFACE
  • PRO_CSYS
Synopsis

/* (In)
The item handle.

ProGeomitemdata** p_data_ptr
/* (Out)

)
Returns
