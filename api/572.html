Object ProDesignatedarea

Function ProDesignatedareaStatusGet


Description
Identifies if the geometry of this Designated Area feature is active, inactive, or out of copied geometry.
Synopsis
#include <ProDesignatedArea.h>
ProErrorProDesignatedareaStatusGet(
ProGeomitem* designated_area_quilt
/* (In)
A quilt from the designated area feature ( or from a data sharing feature referening the designated area ).
*/
ProDesignatedAreaStatus* status
/* (Out)
The designated area status.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
PRO_TK_INVALID_ITEMThe quilt is not from a designated area feature.
PRO_TK_INVALID_TYPEThe quilt is a FLAT designated area feature, which cannot have a non-active status.
