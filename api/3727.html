Function ProSelectionModelitemGet


Description
Gets the model item from a selection object.
Replacement in Object TOOLKIT: pfcSelection::SelModel
pfcSelection::SelItem
Synopsis

ProSelection selection
/* (In)
The selection object.
*/
ProModelitem* p_mdl_item
/* (Out)
The model item.
*/
)
Returns

PRO_TK_NOT_EXISTThe model item doesn't exist.
PRO_TK_BAD_INPUTSThe supplied inputs were incorrect.
