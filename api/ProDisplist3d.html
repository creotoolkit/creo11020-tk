Object ProDisplist3d

Replacement in Object TOOLKIT: DisplayList3D

3-D display list

The 3-D display list contains a set of three-dimensional vectors that represent an approximation to the geometry of the edges of the solid. It gets rebuilt every time the solid is regenerated. This list is used to display three-dimensional entities, such as parts and assemblies.
