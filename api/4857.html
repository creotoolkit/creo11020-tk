Category Custom user interface, Object ProUIList

Function ProUIListSizeSet


Description
Set the size of the list. This field is used only if the parent is a drawing area. If the width or the height parameter is passed a value smaller than the list's minimum size, the function will fail. The list's minimum size can be retrieved using ProUIListMinimumsizeGet.
