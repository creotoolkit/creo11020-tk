Category Manufacturing, Object ProNcseq

Function ProNcseqElemHolesetCsinkdiamSet


Description
Stores the countersink diameter for the specified hole set in a ProNcseqElem data structure. Please note that this routine and the associated PRO_E_HOLES element are provided solely for compatibility with existing Creo Parametric TOOLKIT code. Wherever possible the more powerful PRO_E_HOLESETS element should be used instead.

Applicable Element Id(s):

PRO_E_HOLES
Synopsis
#include <ProNcseqElem.h>

ProNcseqElem elem
/* (In)
The ProNcseqElem handle in which to store the information
*/
int set_nmb
/* (In)
The number of the hole set on which to operate
*/
double csink_diameter
/* (In)
The countersink diameter
*/
)
Returns
PRO_TK_NO_ERRORThe function successfully stored the information.
PRO_TK_BAD_INPUTSOne or more of the arguments are invalid.
