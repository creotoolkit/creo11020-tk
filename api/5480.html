Category Custom user interface, Object ProUITab

Function ProUITabAnchorSet


Description
Sets the position of the tab with respect to a given anchor location. This function is applicable only if the parent of the tab is a drawing area.
Synopsis
#include <ProUITab.h>
ProErrorProUITabAnchorSet(
char* dialog
/* (In)
The name of the dialog that contains the tab.
*/
char* component
/* (In)
The name of the tab component.
*/
int x
/* (In)
X value with respect to the upper left corner of the parent.
*/
int y
/* (In)
Y value with respect to the upper left corner of the parent.
*/
ProUIAnchor anchor
/* (In)
Location on the tab used to set the position.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
