Object ProLayerfilterdata

Imported layer data

Data relating to a layer that is about to be imported from an external file format. This data can be viewed and accessed in Creo Parametric TOOLKIT in the ProIntfimportLayerFilter action.

ProLayerfilterdata Functions:

ProLayerfilterdataActionSet(), ProLayerfilterdataCountsGet(), ProLayerfilterdataNameGet(), ProLayerfilterdataNameSet()
