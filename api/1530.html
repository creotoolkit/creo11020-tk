Category Geometry items, Object ProGeomitem

Function ProGeomitemDiameterEval


Description
Finds the geometry item (surface) diameter. This function is similar to ProSurfaceDiameterEval().
Replacement in Object TOOLKIT: wfcWSelection::EvaluateDiameter
Synopsis
#include <ProGeomitem.h>
ProErrorProGeomitemDiameterEval(
ProSelection p_selection
/* (In)
The geometry item. Only revolved surfaces (cylinder, cone, and so on) are valid.
*/
double* p_result
/* (Out)
The diameter.
*/
)
Returns
PRO_TK_NO_ERRORThe function successfully evaluated the diameter.
PRO_TK_BAD_INPUTSThe input argument is invalid.
