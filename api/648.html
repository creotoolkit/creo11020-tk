Category 3D drawings and annotations, Object ProDimension

Function ProDimensionOrdinatestandardGet


Description
Obtains the ordinate standard used by the given dimension. All of the related ordinate dimensions in a set must use the same standard.
Licensing Requirement
TOOLKIT for 3D Drawings
Synopsis
#include <ProDimension.h>
ProErrorProDimensionOrdinatestandardGet(
ProDimension* dimension
/* (In)
The dimension.
*/
ProDimOrdinateStandard* ordinate_standard
/* (Out)
The ordinate standard.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
