Object ProEdgedata

Replacement in Object TOOLKIT: EdgeDescriptor

Edge data

Information about an edge such as its unique identifier, ID of the surfaces on either side of the edge, edge directions on the surface, array of UV points on the surfaces, edge UV curves on the surfaces, and the curve data handle in the form of ProCurvedata structure.
