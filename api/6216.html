Category Windchill servers, Object ProServer

Function ProServerVersionGet


Description
Obtains the major version of the Windchill server e.g. 7, 6, X05
Replacement in Object TOOLKIT: pfcServerLocation::Version
Synopsis
#include <ProWTUtils.h>
ProErrorProServerVersionGet(
wchar_t* server
/* (In)
A server codebase url or an alias
*/
wchar_t** server_version
/* (Out)
The server version
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
PRO_TK_E_NOT_FOUNDThe server couldn't be found at the given location.
PRO_TK_UNSUPPORTEDThe server is not a Windchill server
