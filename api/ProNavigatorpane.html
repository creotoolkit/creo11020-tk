Object ProNavigatorpane

Custom Panes in the Navigation area

These functions enable Creo Parametric TOOLKIT applications to add custom panes to the Creo Parametric Navigation area. A custom pane can contain either custom dialog box components or WEB pages.

ProNavigatorpane Functions:
