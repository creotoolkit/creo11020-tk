Category Custom user interface, Object ProUIInputpanel

Function ProUIInputpanelWidestringGet


Description
Reports the wide string value of the input panel, if the input type is PROUIINPUTTYPE_WSTRING.
Synopsis
#include <ProUIInputpanel.h>
ProErrorProUIInputpanelWidestringGet(
char* dialog
/* (In)
The name of the dialog that contains the input panel
*/
char* component
/* (In)
The name of the input panel component
*/
wchar_t** value
/* (Out)
The wide string value of the input panel. Free this argument with ProWstringFree()
*/
)
Returns
PRO_TK_NO_ERRORThe function was successful
PRO_TK_GENERAL_ERRORThe function failed
