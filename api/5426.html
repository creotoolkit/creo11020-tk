Category Custom user interface, Object ProUISpinbox

Function ProUISpinboxSizeSet


Description
Set the size of the spinbox. This field is used only if the parent is a drawing area. If the width or the height parameter is passed a value smaller than the spinbox's minimum size, the function will fail. The spinbox's minimum size can be retrieved using ProUISpinboxMinimumsizeGet.
Synopsis

char* dialog
/* (In)
The name of the dialog that contains spinbox.
*/
char* component
/* (In)
The name of the spinbox component.
*/
int width
/* (In)
Width value
*/
int height
/* (In)
Height value
*/
)
Returns

PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
