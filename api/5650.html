Category Custom user interface, Object ProUITable

Function ProUITableSizeSet


Description
Set the size of the table. This field is used only if the parent is a drawing area. If the width or the height parameter is passed a value smaller than the table's minimum size, the function will fail. The table's minimum size can be retrieved using ProUITableMinimumsizeGet.
