Function ProMdlObjectdefaultnameGet


Description
Returns the next available default name for given object type
Replacement in Object TOOLKIT: wfcWModel::GetDefaultName
Synopsis

ProType object_type
/* (In)
only these types are allowed:

  • PRO_REPORT
  • PRO_2DSECTION
  • PRO_3DSECTION
  • PRO_LAYOUT
  • PRO_DWGFORM
  • PRO_MARKUP
*/
ProLine name
/* (Out)
resulting name.
*/
)
Returns
