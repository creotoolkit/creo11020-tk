Object ProEllipsedata

Ellipse data

Superobjects:

ProCurvedata

Ellipse data structure is one of the types of datum curves contained by the ProCurvedata structure. It consists of the type, center of the ellipse, first (x) axis vector of the ellipse, axis vector which is normal to the x axis vector, radii in the direction of 'x_axis' and 'y_axis', and starting angle and end angle (in radians) for the ellipse.
