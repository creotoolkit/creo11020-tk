Category Custom user interface, Object ProUITree

Function ProUITreeIsActivateonreturnEnabled


Description
Identify if the tree is set to generate an activate action by pressing the RETURN key while the tree is in focus.
Synopsis
#include <ProUITree.h>
ProErrorProUITreeIsActivateonreturnEnabled(
char* dialog
/* (In)
The name of the dialog that contains the tree.
*/
char* component
/* (In)
The name of the tree component.
*/
ProBoolean* state
/* (Out)
PRO_B_TRUE when the press of RETURN key enabled; PRO_B_FALSE when the press of RETURN key disabled.
*/
)
Returns
PRO_TK_NO_ERRORThe function succeeded.
PRO_TK_GENERAL_ERRORThe function failed.
PRO_TK_BAD_INPUTSOne or more input arguments was invalid.
