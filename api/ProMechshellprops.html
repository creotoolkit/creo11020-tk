Object ProMechshellprops

Mechanica shell properties

Superobjects:

ProMechitem

A ProMechanica shell property item contains the definition for how the shell geometry should be calculated, for example to create shells that are not homogeneous, or to create shells that are comprised of several layers. A shell property can be assigned to a face, region, or datum surface.
