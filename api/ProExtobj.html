External object

Superobjects:

ProModelitem

External objects are objects created by an application that is external to Creo Parametric. Although these objects can be displayed and selected within a Creo Parametric session, they can not be independently created by Creo Parametric. Using Creo Parametric TOOLKIT functions, you can define and manipulate external objects, which are then stored in the model database.
