Object ProEdge

Edge

This object is an attribute of the following objects:

ProContour, ProGeomitem

An geometric edge of a Creo Parametric part. A set of edges forms a contour, which in turn is part of a surface. Each edge has two surfaces adjacent to it. Edges define the extents of a surface.
